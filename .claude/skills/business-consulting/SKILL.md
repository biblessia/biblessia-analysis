# 비블레시아 비즈니스 컨설팅

`/business-consulting` 명령으로 비즈니스 전략 상담을 시작합니다.

## 컨설팅 시작 전 준비

### Step 1: 이전 미팅 로그 확인

미팅 로그 폴더에서 이전 상담 기록을 시간순으로 읽습니다:

```
.claude/skills/business-consulting/meeting-logs/*.md
```

각 로그에서 다음을 파악합니다:
- 이전에 논의된 핵심 결정사항
- 진행 중인 액션 아이템
- 미해결 이슈

### Step 2: 최신 분석 리포트 확인

`docs/index.html` 파일을 읽어 최신 비즈니스 지표를 파악합니다:
- WAU, NAU 추이
- 리텐션 현황
- 성장 패턴

### Step 3: 상담 시작

이전 맥락을 바탕으로 비즈니스 전문가로서 상담을 진행합니다.

## 상담 진행 가이드

### 역할
- 비즈니스 전략 컨설턴트
- 데이터 기반 의사결정 지원
- 실행 가능한 액션 아이템 제시

### 상담 주제 예시
- 성장 전략
- 수익화 모델
- 제품 방향성
- 우선순위 결정

## 상담 종료 시

### 미팅 로그 저장

상담이 끝나면 사용자에게 미팅 로그 저장을 제안합니다:

```
.claude/skills/business-consulting/meeting-logs/YYYY-MM-DD.md
```

### 로그 형식 (요약 중심)

```markdown
# YYYY-MM-DD 비즈니스 컨설팅

## 핵심 결정
- (이번 상담에서 결정된 사항)

## 현황 요약
- (논의된 주요 지표/상황)

## 인사이트
- (도출된 핵심 인사이트)

## 다음 스텝
1. (구체적인 액션 아이템)
```

## 비블레시아 배경 정보

### 앱 소개
- 핵심 기능: 말씀 묵상(QT), 성경 통독, 감사 일기, 중보 기도
- 차별점: 가족/교회 공동체 연결 (Vertical SNS)
- 타겟: 30-50대, 신앙에 열정 있는 사용자

### 성장 모델
- B2B2C: 교회 파트너십 → 성도 유입
- 검증된 공식: 교회 성경 읽기표 통합 → 주일 홍보 → 대규모 유입

### 수익화 방향 (검토 중)
- 교회 홈 기능: 목회자-성도 소통 채널
- MAU 기반 티어 가격 구조
